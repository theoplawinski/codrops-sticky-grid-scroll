# Sticky Grid Scroll

A structured scroll-driven image grid where movement unfolds progressively within a sticky layout.

![Final result](screenshot.jpg)

[Article on Codrops](https://tympanus.net/codrops/?p=106424)

[Demo](https://tympanus.net/Tutorials/StickyGridScroll)

## Requirements

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

## Default tools

- [Gsap](https://gsap.com/)
- [Lenis](https://github.com/darkroomengineering/lenis)
- [Vite](https://vite.dev/guide/)

## Installation

Clone the repository:

```
git clone git@github.com:theoplawinski/codrops-sticky-grid-scroll.git
```

Go to the root:

```
cd codrops-sticky-grid-scroll
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```
